### 架構

1. node_modiles/         :node 相關套件(隱藏目錄)
2. dist/                 :存放打包後的檔案
3. public/               :公開檔案的目錄
    - favicon.ico
    - index.html
4. src/                  :原始碼目錄  主要開發在這
    - assets/            :靜態檔案預設目錄
    - components/        :元件檔案的預設目錄
    - views/
    - App.vue           
    - main.js           :Vue專案的預設進入點
